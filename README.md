## Features

- 📄 Simple front / back card (with <kbd>**#card**</kbd>)
- 🃏 Custom note types / fields (with <kbd>**#card-_\<custom>_**</kbd>)
- 🛠️ Custom **card templates**
- 🏷️ Global and local **tags**
- 🖼️ Support for **images**
- 🎤 Support for **audios**
- 💻 Support for **code syntax highlight**
- 🟰 Support for **math**
